# Shop

Simple e-commerce platform which includes list of products and a cart. This project was generated with [Angular CLI](https://github.com/angular/angular-cli) version 6.2.4.

## Technologies used
* Angular 6
* SCSS
* Typescript
* RxJS

## Development server

To start the json-server run `npm run api`. Run `ng serve` for a dev server. Navigate to `http://localhost:4200/`. The app will automatically reload if you change any of the source files. 

## Running unit tests

Run `ng test` to execute the unit tests via [Karma](https://karma-runner.github.io).

## Further help

To get more help on the Angular CLI use `ng help` or go check out the [Angular CLI README](https://github.com/angular/angular-cli/blob/master/README.md).
